## Arquivos principais
- `index.html`
- `assets/css/style.css`
- `assets/js/main.js`
- `assets/img/`

## Melhorias incluídas
- Novo hero premium com proposta de valor forte
- Menu responsivo corrigido
- Portfólio expandido com novos nichos: advocacia, odontologia, arquitetura, imóveis e carros
- Cards de serviços, método, planos, depoimentos, FAQ e contato
- Formulário integrado ao WhatsApp
- Animações de scroll reveal
- Contadores animados
- Tema claro/escuro
- SEO básico e Schema markup

## Observação
Algumas imagens dos novos modelos usam links externos do Unsplash. Se quiser deixar 100% offline, baixe essas imagens e salve em `assets/img/`, depois troque os caminhos no `index.html`.
